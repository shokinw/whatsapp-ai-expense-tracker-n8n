# AI Sales Intelligence Automation System

## Overview
This project automates sales intelligence reporting using AI and APIs.

## Workflow
- Google Sheets trigger
- News API fetch
- AI analysis (OpenAI)
- HTML report generation
- Gmail email automation
- Google Calendar scheduling

## Tech Stack
n8n, OpenAI, NewsAPI, Google APIs, HTML

## How to Use
1. Import workflow.json into n8n
2. Add API keys
3. Connect Google Sheets
4. Run workflow

## Output
Automated sales briefing sent to email + calendar event created.
